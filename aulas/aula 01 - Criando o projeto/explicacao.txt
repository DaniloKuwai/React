Nessa aula vamos iniciar a criação do projeto 

1º -> Escolher a pasta que vai ser iniciado o projeto
2º -> dentro da pasta dar o seguinte comando:

npx create-react-app fundamentos-react

-> o npx serve para criar um projeto, sem a necessidade de instalar um pacote global e sim temporário
apenas para criar o projeto.
-> o create-react-app serve para criar o projetoo
-> o fundamentos-react é o nome do meu projeto

-------------------------------

Após o projeto basta entrar na pasta fundamentos-react e dar o comando npm start para iniciar o projeto
ele vai startar o projeto e vai abrir a página inicial

-------------------------------

Vamos apagar todos arquivos dentro da pasta src para iniciar do zero as aulas desde o começo, vamos
apenas criar uma index.js vazia dentro de src para iniciar
